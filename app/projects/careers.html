Careers

Responsive landing page for career opportunities at Metaps Inc.
Overview

Design and develop a landing page which focuses on guiding the user through the company's key business fundamentals and current industry performance. Each section must be unique, identifiable and attuned to the brand experience.

Process & Involvement
  • Wireframe and prototype creation
  • Art direction and photography
  • UI design, branding, icon curation
  • Front-end development and debugging
Focus Point

Utilise a templating language where the principles of writing DRY (don't repeat yourself) code are implemented throughout. Additionally, validate forms in real-time to provide an intuitive user experience.

Focus Solution

Implement AngularJS to share information between partials and provide instant feedback to the user on correct and incorrect information.

OfferBoard

Responsive landing page for app monetization product
Overview

To design and develop a landing page which instantly informs the user about key product features whilst providing appropriate call-to-action dialogue. The site must incorporate an integrated brand experience and provide solid reassurance about the product.

Process & Involvement
  • Wireframe and prototype creation
  • Art direction and branding
  • UI design and icon curation
  • Front-end development and debugging
Focus Point

Provide an intuitive integration flow which leads the users through easy-to-understand steps.

Focus Solution

Use icon-header blocks of numbered content to take the user through three simple steps. Add a speech border to focus attention on integration steps order.
